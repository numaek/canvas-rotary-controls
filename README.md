# Canvas Drehregler

Some examples for different canvas rotary controls.

- percent value
- step controller
- endless controller
- middle position
- different scales (1 10, -10 0, -5 +5, ...)
- different value history (linear, logarithmic degressive and progressive)
- illuminated scale
- toggle switch
